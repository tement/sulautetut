# Sulautetut järjestelmät - väkiluku
Olen Varian opiskelija. Tämä on harjoitustyö.

Projekti käyttää Node.js:ää. Se riippuu näistä kirjastoista:
- react-router-dom
- victory
Kirjastot täytyy ladata npm install -komennon avulla, jotta ne ovat käytettävissä.
